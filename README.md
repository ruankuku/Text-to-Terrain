# Text-to-Terrain: A Data-Driven Topographical Visualization of the Roman Empire (AD 180–337)

## Overview

Text-to-Terrain is a digital humanities project that transforms historical narratives into data-driven 3D landscapes.
The project focuses on the period AD 180–337 of the Roman Empire, drawing from Gibbon’s The Decline and Fall of the Roman Empire and converting qualitative historiography into a structured, spatial, and interactive visual model.

The workflow integrates text processing, event quantification, Rhino modeling, AI-based image enhancement, 3D reconstruction, and TouchDesigner interaction into a complete pipeline.

This repository contains the full implementation, source files, processing scripts, and interactive components used to build the final visualization.

## Core Tools

Python 3.12 — automation, text processing, data validation

LLM Models — summarization, theme extraction, event extraction

Rhino 6 + Grasshopper — geometric modeling

ComfyUI — image enhancement

NanoBanana — terrain texture refinement

Hunyuan 3D / Hy3D — mesh reconstruction and UV baking

Nomad Sculpt — mesh cleanup and symbolic editing

TouchDesigner — final interactive system
